\section{Численные методы решения обыкновенных дифференциальных уравнений}
\subsection{Решение задачи Коши методом Эйлера}
Рассмотрим следующую задачу "--- однородное дифференциальное уравнение первого порядка
\begin{equation*}
\begin{cases}
 y' = f(x, y) &\text{y = y(x)}\\
 y(x_0) = y_0
\end{cases}
\end{equation*}
Она может быть решена методом Эйлера.

Пусть y(x) = $x^3 + x + 1$

\textbf{Решение:}
